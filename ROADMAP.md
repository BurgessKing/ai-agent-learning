# 项目目标与路线图

> **最后更新**：2026-07-17 | **状态**：第一阶段 — 学习 + 本地实验

---

## 🎯 核心目标

通过 **1-2 个月**的系统学习与实践，掌握大模型与 Agent 核心技术，将 AI Agent 融入「易客拉公交出行平台」项目，实现：

1. **本地运行调试** — Agent 在 Windows 本机跑通全链路
2. **线上问题模拟与处理** — 覆盖超时/幻觉/并发冲突/成本失控等生产问题
3. **沉淀可复用的 Agent 框架** — 不是一次性 Demo，而是可迭代的工程方案

---
